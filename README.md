# Open-screen-animation
类手机应用的开屏动画，非线性动画，基于scratch 3，将sb3文件转化为HTML，最大60fps
<br><br>

## 使用说明 
下载该目录下的“开屏动画.html”文件，里面包括图片，所以该文件体积较大。下载后直接运行。图标可拖动，点击图标会有开屏动画，然后就啥也没有了。 <br>
**空格键**关闭当前开屏
